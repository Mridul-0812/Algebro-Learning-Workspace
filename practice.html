Nested Brackets

NOTE: You always start with round brackets ( ) for the first, inner grouping, then use square brackets [ ] if you need a second set outside of them (as in the equation above). Essentially, the innermost brackets or the first set of brackets are always round.
💡 PRO TIP: This is a challenge but it is exactly what you have learnt before. Just expand the brackets twice (first the round ones, then the square ones). Work from the inside out before you multiply everything by 4!
