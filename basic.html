Example 2: Balance the Seesaw

LHS has 5 blocks. Drag the correct number of blocks to the RHS to balance!

Equation: 5 = 𝑥 + 1

(Not Balanced)

LHS
▲
RHS

Steps:

1) In this example, LHS has five (5) blocks and the RHS already has one (1) block.
2) We need to add 𝑥 blocks to the RHS to balance both sides.
3) According to this idea, to form LHS = RHS, we need to form the equation 5 = 𝑥 + 1.
4) To solve such an equation, the idea is to make 𝑥 the subject of the equation (in other words, solve for 𝑥 by removing the +1 on the RHS).
5) We can minus 1 from the RHS. However, whatever is done to the RHS should be done to the LHS as both sides must be balanced.
6) Thus, we attain 5 - 1 = 𝑥 + 1 - 1. It equals 4 = 𝑥 OR 𝑥 = 4.
7) Hence, we need to add four (4) blocks.
